✅ Implementované změny

1. Tlačítko "Opustit hru"
- Přidáno červené tlačítko "Opustit hru" do herních ovládání
- Zobrazuje potvrzovací dialog před opuštěním
- Vrací na hlavní menu a resetuje hru
- Zachovává síň slávy
2. Síň slávy - odstranění mazání
- Odstraněno tlačítko "Vymazat vše" z modalu síně slávy
- Záznamy zůstávají trvale uložené
- Nelze smazat historii výsledků
3. Automatické zobrazení síně slávy
- Po uložení skóre se automaticky zobrazí síň slávy
- Uživatel okamžitě vidí svůj nový záznam
- Zpoždění 500ms pro plynulý přechod

🔧 Technické detaily

CSS styly: Přidána třída .btn-danger pro červené tlačítko
HTML: Tlačítko "Opustit hru" přidáno do .roll-controls
JavaScript: Funkce quitGame() s potvrzovacím dialogem
Event listenery: Napojení na quitGameBtn
Hall of Fame: Auto-zobrazení po saveScore()

🎯 Návod k testování

  1. Spusťte hru kliknutím na "Začít hru"
  2. Ověřte, že se zobrazilo červené tlačítko "Opustit hru"
  3. Klikněte na "Opustit hru" a potvrďte dialog
  4. Ověřte návrat na hlavní menu
  5. Spusťte hru a dohraje ji do konce
  6. Jako vítěz zadejte podpis a uložte skóre
  7. Ověřte automatické zobrazení síně slávy
  8. Zkontrolujte, že v síni slávy není tlačítko "Vymazat vše"
