## 🎮 HERNÉ PRAVIDLÁ

- **Cieľ:** Prvý na target score (default 10000)
- **Farkle:** Žiadne bodujúce kostky = koniec tahu s 0 bodmi
- **Trojice:** 3x1=1000pts, 3x2=200pts, 3x3=300pts, atď.
- **Jedničky:** 100pts each (mimo trojice)
- **Päťky:** 50pts each (mimo trojice)
